# SeaBattle

The position of the ships is randomly selected. There are player's ship and computer's shots on the top. The player’s shot field is below. When hit, the square turns red, otherwise blue. Enemy's shots are randomly selected. Also player can look into the opponent’s position by clicking on "Spy on an adversary" (2nd screenshot).
